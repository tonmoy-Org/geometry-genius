1. What is the DOM ?

DOM is an acronym that stands for Document Object Model.The DOM object is automatically created by the web browser itself upon opening an HTML web page.It provides a way for developers to access and modify the elements and content of a web page dynamically, allowing for the creation of interactive web applications. With the DOM, developers can add, remove, or modify elements and their attributes in real-time.Overall, the DOM is an essential tool for web developers, providing a way to create interactive and dynamic web applications that can respond to user input and events.

3. What is event bubble

Event bubbling is a way of event propagation in the HTML DOM. It allows you to attach a single event listener to a parent element instead of each child element individually.

4. What is Callback function and how it's work

A callback function is a function that is passed as an argument to another function.The purpose of a callback function is to allow the calling function to continue execution without waiting for the callback function to complete.
